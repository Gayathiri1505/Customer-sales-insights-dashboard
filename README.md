# Customer Sales & Insights Dashboard

## 📌 Project Overview
This project presents an interactive **Customer Sales & Insights Dashboard** developed to analyze customer purchasing behavior, revenue trends, seasonal spending patterns, and demographic insights.

The dashboard helps businesses understand:
- Customer distribution
- Revenue generation
- Product category performance
- Seasonal spending behavior
- Purchase trends by location
- Customer review ratings

The project combines:
- **Power BI** for dashboard visualization
- **Python** for data preprocessing and EDA
- **SQL** for querying and analysis
